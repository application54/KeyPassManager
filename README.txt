Менеджер паролей v1.0.0

Описание:
Безопасное приложение для хранения и управления паролями с шифрованием данных.

Особенности:
- Шифрование данных с использованием мастер-пароля
- Генератор безопасных паролей
- Хранение паролей в зашифрованной базе данных
- Копирование паролей и логинов в буфер обмена
- Портативная версия (не требует установки)

Использование:
1. Запустите файл "Менеджер паролей.exe"
2. При первом запуске установите мастер-пароль
3. Используйте приложение для хранения паролей

Безопасность:
- Все пароли шифруются перед сохранением в базу данных
- Мастер-пароль не хранится нигде, запомните его!
- Без мастер-пароля невозможно восстановить данные

Требования:
- Windows 7/8/10/11
- .NET Framework 4.5 или выше (обычно уже установлен)

Примечание:
Никогда не передавайте свой мастер-пароль третьим лицам.
Регулярно создавайте резервные копии файла passwords.db.
